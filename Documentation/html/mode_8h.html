bool askSkipLongAlgorithms ( )

Asks the user whether to skip long-running algorithms.

This function prompts the user with a yes/no question about skipping time-consuming algorithms (e.g., exhaustive search on large datasets). It continues to prompt until a valid input is provided.

Acceptable inputs (case-insensitive):

  • "yes", "y": returns true
  • "no", "n": returns false
Returns
true If the user chooses to skip long-running algorithms.
false If the user chooses to include all algorithms.

Prompts the user to seelct a valid algorithm.

This function continuously prompts the user via standard input until a valid algorithm is entered. Valid algorithms include: "exhaustive", "dynamic programming", "approximation", "genetic", and "benchmark".
