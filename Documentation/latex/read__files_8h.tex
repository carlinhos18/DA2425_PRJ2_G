Reads the truck data from a CSV file and loads it into the provided Algoritm structure. 

This function opens a truck data file, parses its content to retrieve the truck\textquotesingle{}s capacity and number of pallets, and sets the {\ttfamily Total\+Weight} in the {\ttfamily Algorithm} object. It also determines the corresponding pallets file based on the truck ID and reads its data.
