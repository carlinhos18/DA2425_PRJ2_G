{\footnotesize\ttfamily void benchmark\+Mode (\begin{DoxyParamCaption}\item[{bool}]{skip\+Long }\end{DoxyParamCaption})}



Benchmarks the performance of different algorithms on a series of datasets. 

This function loads multiple dataset files, each representing a truck\textquotesingle{}s pallet configuration, and applies different knapsack-\/solving algorithms to each. It measures and records the execution time for each algorithm and outputs the results both to the console and a file named {\ttfamily "{}benchmark\+\_\+test.\+txt"{}}.

The algorithms benchmarked are\+:
\begin{DoxyItemize}
\item {\ttfamily "{}exhaustive"{}}\+: Brute-\/force approach (may be skipped for larger datasets).
\item {\ttfamily "{}dynamic programming"{}}\+: Optimal DP-\/based solution.
\item {\ttfamily "{}approximation"{}}\+: Greedy heuristic (hybrid of profit-\/to-\/weight ratio and raw profit).
\item {\ttfamily "{}genetic"{}}\+: Genetic Algorithm for approximating solutions.
\end{DoxyItemize}

The benchmarking process involves\+:
\begin{DoxyItemize}
\item Iterating through 10 predefined input files (named {\ttfamily Truck\+And\+Pallets\+\_\+01.\+csv} to {\ttfamily Truck\+And\+Pallets\+\_\+10.\+csv}).
\item Applying each algorithm to the dataset.
\item Timing each algorithm using {\ttfamily std\+::chrono}.
\item Optionally skipping the exhaustive algorithm on medium-\/size datasets if {\ttfamily skip\+Long} is true.
\end{DoxyItemize}


\begin{DoxyParams}{Parameters}
{\em skip\+Long} & if tru, the exhaustive algorithm will be skipped for datasets 4, 5, and 6 to avoid long execution times.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
The function writes benchmark results to a file named {\ttfamily benchmark\+\_\+test.\+txt}, overwriting any previous content. Time is reported in microseconds for fast algorithms and seconds for slower ones. 
\end{DoxyNote}
